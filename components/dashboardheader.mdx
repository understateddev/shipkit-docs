## Usage

<Tabs>
  <Tab title="Astro">
    ```tsx
    ---
    import { DashboardHeader } from '~/components';
    ---
    
    <DashboardHeader user={Astro.locals.user} />
    ```
  </Tab>

  <Tab title="Next.js">
    ```tsx
    import { DashboardHeader } from '~/components';
    import { getCurrentUser } from '~/core/auth/server';

    export default async function Page() {
      const user = await getCurrentUser();

      return <DashboardHeader user={user} />;
    }
    ```
  </Tab>
</Tabs>

## Props

<ResponseField name="user" type="User | null" required>
  The current user. (<a href="#user">User</a>)
</ResponseField>

<ResponseField name="children" type="any">
  A replacement for the avatar in the header.
</ResponseField>

## Content

To update the title in the header, edit `site.name` in `src/config/site.ts`.

```typescript
import { env } from '~/core/env';

export const site = {
  name: 'ShipKit',
  ...
};
```
