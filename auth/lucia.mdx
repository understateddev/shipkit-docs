Lucia offers a straightforward authentication solution within ShipKit, allowing you to easily implement user authentication and session management. It's designed for developers looking for a lightweight, yet secure authentication system that integrates seamlessly with various frameworks.

## Lucia config

The config for Lucia is located at `src/core/auth/server/lucia.ts`.
